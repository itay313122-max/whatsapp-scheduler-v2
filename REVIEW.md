## 4. הוראות להוספת נתונים עתידיים

### הוספת KPI חדש (לשוניות 1 ו-2)

1. פתח את הלשונית הרלוונטית (`📋 סיכום כללי` או `🛠️ מוצר וטכנולוגיה`)
2. הכנס שורה חדשה **לפני** שורת הממוצע (לפני שורת הסיכום `DARK_BLUE`)
3. בעמודה `B` הכנס את ערך הבסיס כמספר (לדוגמה: `65`)
4. בעמודה `C` הכנס נוסחה: `=B<ROW>/100`
5. עדכן את טווח ה-`AVERAGE` בשורת הסיכום כך שיכלול את השורה החדשה

### הוספת דף לקוח פעיל (לשונית 2)

1. עבור לקטע "דפי לקוחות פעילים" בלשונית `🛠️ מוצר וטכנולוגיה`
2. הכנס שורה חדשה **לפני** שורת `=COUNTA`
3. מלא: מספר סידורי, שם הדף, URL, סטטוס
4. ה-`COUNTA` יתעדכן אוטומטית אם הטווח מכסה את השורה החדשה (בדוק שהטווח מתעדכן)

### הוספת חבר צוות (לשונית 3)

1. עבור לקטע "צוות מייסדים" בלשונית `👥 צוות ומשקיעים`
2. הכנס שורה לפני שורת הסיכום
3. מלא: שם, תפקיד, התמחות, ניסיון
4. ה-`COUNTA(A6:A9)` — עדכן את הטווח ל-`COUNTA(A6:A10)` ידנית

### הוספת משקיע / שותף (לשונית 3)

1. עבור לקטע "משקיעים ושותפים"
2. הכנס שורה לפני שורת הסיכום
3. מלא: שם/גוף, תפקיד/קשר
4. עדכן את טווח ה-`COUNTA` בשורת הסיכום

### הוספת ערוץ הכנסה / פרויקט (לשונית 4)

זהה לאמור לעיל — הכנס לפני שורת הסיכום ועדכן טווח `COUNTA` בהתאם.

### עדכון הקובץ דרך הסקריפט
